üîß Pre-Testing Setup

Step 1: Extension Installation Verification

  1. Navigate to chrome://extensions/
  2. Verify ResumeHub extension is enabled
  3. Click the extension icon in the toolbar
  4. Popup should open without errors
Expected: Popup opens showing all interface cards (API Key, Resume Upload, etc.)

Step 2: Console Monitoring Setup

  1. Open Chrome DevTools (F12)
  2. Go to Console tab
  3. Clear console and monitor for errors during testing
Expected: No error messages in console during normal operation

üé® User Interface Testing

Theme Toggle Test

  1. Locate theme toggle switch in header
  2. Click to switch from light to dark mode
  3. Verify background and text colors change
  4. Click again to switch back to light mode
  5. Reload extension popup
  6. Verify theme preference is saved

  1. Click on each card header (API Key, Resume Upload, etc.)
  2. Verify cards expand and collapse
  3. Test clicking toggle arrows

üîë API Key Management Testing

API Key Input and Validation

  1. Click on "API Key Configuration" card to expand
  2. Enter a valid Google Gemini API key
  3. Press Tab or click outside input field
  4. Verify "Saved" status message appears
  5. Verify card collapses automatically
  6. Reload popup and verify key is remembered

Invalid API Key Handling

  1. Enter an invalid or empty API key
  2. Try to use features requiring API key
  3. Verify appropriate error messages

üìÑ Resume Upload Testing

File Upload Process

  1. Click on "Resume Upload" card to expand
  2. Click "Choose File" button
  3. Select a resume file (PDF, DOC, DOCX, or TXT)
  4. Verify file name appears
  5. Verify "Using: [filename]" status message
  6. Test "Clear Resume" button
  7. Test "Download Original" button

Drag and Drop Upload

  1. Drag a resume file over the resume upload card
  2. Verify visual feedback (highlight/border change)
  3. Drop the file
  4. Verify file is processed same as manual upload
Expected: Visual drag feedback and successful file processing

Invalid File Handling

  1. Try uploading unsupported file types (e.g., .jpg, .exe)
  2. Try uploading files larger than 10MB
  3. Verify appropriate error messages
Expected: Clear error messages for invalid files

üîß Troubleshooting Guide

If Tests Fail:

  1. Reload Extension: Go to chrome://extensions/, disable and re-enable ResumeHub
  2. Check Console: Look for JavaScript errors in browser console (F12)
  3. Verify Files: Ensure all extension files are present and not corrupted
  4. Clear Storage: Clear extension storage and re-enter API key
  5. Test in Incognito: Try testing in incognito mode to rule out conflicts
  6. Update Chrome: Ensure Chrome browser is up to date

Common Issues and Solutions:

Issue Possible Cause Solution
Popup won't open Extension not loaded properly Reload extension in chrome://extensions/
API operations fail Invalid API key or quota exceeded Check API key and billing status
File upload fails File too large or unsupported format Use supported formats under 10MB
Auto-fill doesn't work No form fields detected Ensure page has fillable form fields
Theme doesn't persist Storage permission issue Check extension permissions
